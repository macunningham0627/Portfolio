Minesweeper

Minesweeper project

Minesweeper is a game in which the player attempts to reveal squares in a grid while avoiding mines.

A square is revealed by clicking on it; this will then show a number indicating how many adjacent squares contain a mine. Players must use this information to determine which squares are safe.

The player wins when every safe square has been revealed.

The player loses if a square with a mine is clicked on.

Flagging is an additional feature that allows the player to mark squares that they believe to contain mines.

The locations of mines are randomized every time the game is played, creating a new experience each time. This project uses the Java programming language to build a fully functioning version of Minesweeper. Visit the link to try it out.
